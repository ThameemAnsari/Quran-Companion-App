Our Solution

API Integration

Powered by Quran Foundation APIs

Endpoint Purpose in the App
GET /api/v4/search?q=… Mood-based Ayah discovery — keywords map to emotional themes
GET /api/v4/verses/by_chapter/{id} Fetches all verses with Uthmani Arabic + Translation ID 20
GET /api/v4/chapters/{id} Surah name and metadata for display throughout the app
cdn.islamic.network/…/ar.alafasy Mishary Alafasy MP3 recitation streamed per verse globally
Translation ID 20 Sahih International English translation rendered under every Ayah

All calls made via src/services/quranApi.ts — no API key required.
